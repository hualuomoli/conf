# 配置

## 修改npm的镜像地址

    文件位置:C:\Users\电脑用户名\.npmrc
    内容
        registry = http://registry.npm.taobao.org/
        loglevel = http

## 全局安装插件
npm install -g npm bower

    npm             后端包管理器
    bower           前端包管理器
    

## gulp配置
npm install -g gulp gulp-cli

    gulp            构建工具
    gulp-cli        command-line interface
