# 用户

## 添加用户
```
useradd
    -d 用户登入时的主目录(默认值/home/<用户名>)
    -g 用户所属的群组(默认值为100，即users)
    -s 用户登入后所使用的shell(默认值为/bin/bash)
    -c 加上备注文字，备注文字保存在passwd的备注栏中。
    -f 密码过期后多少天即关闭该账号(0:立即被停用,-1:永久可用.默认值为-1)
    -e 账号的失效日期，日期格式为MM/DD/YY。缺省表示永久有效。
    用户名
```

## 修改密码
`password 用户名`

## 删除用户
```
userdel
    用户名
```
