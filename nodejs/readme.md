
# 更新NPM
`npm install -g npm`

# 全局安装

## 前端包管理器
`npm install -g bower`

## 构建工具
`npm install -g gulp gulp-cli`

## 移动开发
`npm install -g cordova ionic`

## RAML API
`npm install -g api-designer raml2html`

## Express
`npm install -g express express-generator`

## SCSS
`npm install -g node-sass`
`npm install postcss-cli autoprefixer`
